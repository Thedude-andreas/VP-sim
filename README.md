# VP-simulator

En fristående webbaserad procedur-simulator för grundläggande VP-träning i Aviat Husky A1B-kontext.

Öppna `index.html` direkt i en webbläsare. Ingen build, server, deploy eller extern tjänst krävs.

## Första versionens fokus

- Trottel, propellerreglage, magringsreglage och förgasarvärme.
- Airspeed, altimeter, manifold pressure, RPM och EGT.
- Förenklad vy genom vindrutan med bana, horisont, fart- och höjdkänsla.
- Procedurstöd för takeoff, stigning, cruise och approach.
- Motorljud som aktiveras med ljudknappen och följer varvtalet.

Flygmodellen är avsiktligt förenklad för procedural träning, inte för exakt flygdynamik.
